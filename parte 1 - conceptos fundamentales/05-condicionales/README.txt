Objetivo:

- Aprender a usar las estructuras condicionales `if`, `if-else` y `if-else if-else`.
- Comprender cómo permiten que un programa tome decisiones.

Concepto clave:

- Una condición se evalúa como verdadera (distinta de 0) o falsa (igual a 0).
- Los condicionales controlan qué bloque de código se ejecuta.
- Las decisiones en cascada se gestionan con `else if`.

Instrucciones:

- Compilar: gcc -Wall -Wextra -std=c11 -o condicional 05_condicionales.c
- Ejecutar: ./condional (Linux/macOS) ./condicional.exe (Windows)
- Introducir números cuando se solicite.

Preguntas de reflexión:

- ¿Qué ocurre si no se cumple ninguna condición?
- ¿Qué diferencia hay entre `=` y `==`?
- ¿Por qué se evalúan las condiciones de arriba hacia abajo?
- ¿Cómo cambia el flujo del programa con `else`?
- ¿Qué ocurre si eliminas un paréntesis o una llave?

Katas rápidas:

- Kata 1: Cambia la condición para detectar si un número es múltiplo de 3.
- Kata 2: Añade un mensaje para cuando la nota sea superior a 95.
- Kata 3: Usa `&&` para comprobar si un número está entre 10 y 20.

Prompt IA:

"Explícame paso a paso cómo funcionan las sentencias if, if-else y else-if en este código. Señala posibles errores comunes con el operador == o las llaves. Propón una variante con condiciones anidadas sin cambiar los nombres de las variables."
